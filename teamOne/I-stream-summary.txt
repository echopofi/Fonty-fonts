I-STREAM MOVIE PROJECT SUMMARY
=============================

PROJECT STRUCTURE
----------------
Root: /home/pofii/Desktop/Blockfuse/javascript/teamOne/movieProject/I-stream/

Key Files:
- build/index.html      Main streaming page
- build/login.html  Login/Register page
- build/js/main.js   Movie data catalog (5 sample movies)
- build/js/app.js    Main app logic (auth, grid, player, search)
- build/js/login.js  Authentication logic
- build/css/style.css Tailwind + custom styles
- package.json      Node config (Tailwind CLI, Prettier)

TECH STACK
----------
- Vanilla HTML/CSS/JavaScript (no framework)
- Tailwind CSS v3.4 (via CDN)
- Syne + DM Sans fonts
- LocalStorage for auth persistence

CURRENT FEATURES
---------------
1. Authentication
   - Email/password login & registration
   - Guest access mode
   - Session handling via LocalStorage

2. Main Page (index.html)
   - Fixed navbar with scroll effect
   - Search (desktop + mobile)
   - Genre filtering (pills)
   - Sort options (A-Z, rating, year)
   - Responsive movie grid (2-5 columns)
   - Hero banner with featured movie
   - Detail modal with movie info
   - Custom video player with controls

3. Video Player
   - Play/pause, volume, mute
   - Progress bar with seek
   - Skip -10s / +10s
   - Fullscreen toggle
   - Keyboard shortcuts

4. Login Page
   - Split layout (left branding, right form)
   - Tab switching (Sign In / Register)
   - Password show/hide toggle
